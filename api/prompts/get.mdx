---
title: Get Prompt
api: GET /v1/prompts/:owner/:name
---

Retrieve a specific prompt by owner username and prompt name.

## Request

### Path Parameters

<ParamField path="owner" type="string" required>
  The username of the prompt owner.
</ParamField>

<ParamField path="name" type="string" required>
  The unique name of the prompt.
</ParamField>

## Response

<ResponseField name="prompt" type="object">
  The prompt object containing all prompt details.

  <ResponseField name="prompt.id" type="string">
    The unique identifier (UUID) for the prompt.
  </ResponseField>

  <ResponseField name="prompt.name" type="string">
    The name of the prompt.
  </ResponseField>

  <ResponseField name="prompt.description" type="string">
    The description of the prompt.
  </ResponseField>

  <ResponseField name="prompt.owner_id" type="string">
    The UUID of the user who owns the prompt.
  </ResponseField>

  <ResponseField name="prompt.created_at" type="string">
    ISO 8601 timestamp of when the prompt was created.
  </ResponseField>

  <ResponseField name="prompt.owner_username" type="string">
    The username of the prompt owner.
  </ResponseField>

  <ResponseField name="prompt.tags" type="string[]">
    Array of tags associated with the prompt.
  </ResponseField>
</ResponseField>

## Examples

<CodeGroup>

```bash cURL
curl https://api.prompts.dev/v1/prompts/alice/code-review-assistant
```

```javascript JavaScript
const response = await fetch(
  'https://api.prompts.dev/v1/prompts/alice/code-review-assistant'
);

const data = await response.json();
const prompt = data.prompt;
```

```python Python
import requests

response = requests.get(
    'https://api.prompts.dev/v1/prompts/alice/code-review-assistant'
)

data = response.json()
prompt = data['prompt']
```
