---
title: Search Prompts
api: GET /v1/prompts
---

Search for prompts using a query string with optional pagination.

## Request

### Query Parameters

<ParamField query="q" type="string" required>
  The search query string. This parameter is required.
</ParamField>

<ParamField query="limit" type="integer" default="20">
  Maximum number of results to return. Must be greater than 0. Defaults to 20.
</ParamField>

<ParamField query="offset" type="integer" default="0">
  Number of results to skip for pagination. Must be greater than or equal to 0. Defaults to 0.
</ParamField>

## Response

<ResponseField name="items" type="array">
  Array of prompt objects matching the search query.

  <ResponseField name="items[].id" type="string">
    The unique identifier (UUID) for the prompt.
  </ResponseField>

  <ResponseField name="items[].name" type="string">
    The name of the prompt.
  </ResponseField>

  <ResponseField name="items[].description" type="string">
    The description of the prompt.
  </ResponseField>

  <ResponseField name="items[].owner_id" type="string">
    The UUID of the user who owns the prompt.
  </ResponseField>

  <ResponseField name="items[].created_at" type="string">
    ISO 8601 timestamp of when the prompt was created.
  </ResponseField>

  <ResponseField name="items[].owner_username" type="string">
    The username of the prompt owner.
  </ResponseField>

  <ResponseField name="items[].tags" type="string[]">
    Array of tags associated with the prompt.
  </ResponseField>
</ResponseField>

<ResponseField name="query" type="string">
  The search query that was executed (echoed back).
</ResponseField>

<ResponseField name="limit" type="integer">
  The limit that was applied to the search results.
</ResponseField>

<ResponseField name="offset" type="integer">
  The offset that was applied to the search results.
</ResponseField>

## Examples

<CodeGroup>

```bash cURL
curl "https://api.prompts.dev/v1/prompts?q=code+review&limit=10&offset=0"
```

```javascript JavaScript
const params = new URLSearchParams({
  q: 'code review',
  limit: '10',
  offset: '0'
});

const response = await fetch(
  `https://api.prompts.dev/v1/prompts?${params}`
);

const data = await response.json();
const prompts = data.items;
```

```python Python
import requests

response = requests.get(
    'https://api.prompts.dev/v1/prompts',
    params={
        'q': 'code review',
        'limit': 10,
        'offset': 0
    }
)

data = response.json()
prompts = data['items']
```

</CodeGroup>

<Note>

  For example: first request uses `offset=0`, second request uses `offset=10`, third uses `offset=20`, etc.
</Note>
